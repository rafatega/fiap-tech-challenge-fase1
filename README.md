# fiap-tech-challenge-fase1
Criação de uma API Pública para Consulta de Livros. Este repositório contém a solução para o desafio de construir uma infraestrutura completa de dados para um sistema de recomendação de livros.

# Clonar o repositório
Clone o repositório usando o comando abaixo no terminal, dentro da pasta onde deseja salvar o projeto:
```bash
git clone https://github.com/rafatega/fiap-tech-challenge-fase1.git
```

# Criar venv
Crie um ambiente virtual para o projeto com o comando:
```bash
python -m venv venv
```

# Ativar venv
Ative o ambiente virtual com o comando:
```bash
venv\Scripts\activate
```

# Instalar dependências
Instale as dependências do projeto com o comando:
```bash
pip install -r requirements.txt
```
